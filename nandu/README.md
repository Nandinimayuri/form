HTML Form Practice Project :-

📌 Project Title:-
Complete HTML Form Using All Input Elements and Tags

📖 Description:-
This project is a beginner-friendly HTML form that demonstrates the use of all important form-related tags and input elements. It helps in understanding how user input is collected and structured using HTML forms. The form includes different types of input fields, labels, selection controls, and validation features.

🧩 Features Used in This Form:-
This form includes the following HTML form elements:
- `<form>` – Main form container
- `<input>` – Text, password, email, number, radio, checkbox, file, date, etc.
- `<label>` – For input descriptions
- `<textarea>` – Multi-line text input
- `<select>` and `<option>` – Dropdown list
- `<button>` – Submit and reset buttons
- `<fieldset>` – Groups related fields
- `<legend>` – Title for fieldset
- HTML form attributes like:
  - `required`
  - `placeholder`
  - `value`
  - `name`
  - `type`

🧾 Form Elements Included:-
- Text input (Name, Username)
- Email input
- Password input
- Number input
- Radio buttons (Gender selection)
- Checkbox (Hobbies/Interests)
- Date picker
- File upload
- Dropdown selection (Country/Options)
- Textarea (Address/Comments)
- Submit and Reset buttons

🎯 Purpose:-
The purpose of this project is to practice and understand all HTML form elements and how they work together to collect user data in a structured way.
